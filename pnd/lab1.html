Practical Network Defense

Main Kathara commands (aliases):
- start (restart) a lab: lstart/lrestart
- stop a running lab: lclean
- wipe the kathara environment: kwipe
- list virtual networks and virtual interfaces of VMs: docker network list

IP commands:
-clean IP address of an ethernet interface: ip addr flush ethN;

Guide
Ex.1
A host has to receive 4 main pieces of information:
- IP address
- The Netmask
- The IP address of its default Gateway
- The IP address of a DNS


  1. launch kahara lstart
  2. for pc1:
    - nano /etc/network/interfaces
    - paste these commands:
     auto eth0
     iface eth0 inet static
     address 192.168.100.31/29
     gateway 192.168.100.30
     dns-nameservers 151.100.4.2 151.100.4.13
    - write in the terminal: ifup eht0
  3. For pc2:
    - ifconfig eth0 192.168.100.32
    - ifconfig eth0 netmask 255.255.255.248
    - ifconfig eth0 up
    - nano etc/resolve/conf
    - write nameserver 151.100.4.2
    nameserver 151.100.4.13
